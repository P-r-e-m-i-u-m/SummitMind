# Privacy

SummitMind is local-first.

In the current MVP:

- Memories are stored in your browser's `localStorage`.
- There is no server.
- There are no analytics scripts.
- There are no third-party trackers.
- Nothing is uploaded unless you add that feature yourself.

## Important Notes

Browser storage is convenient, but it is not a secure vault. Anyone with access to your browser profile may be able to read the data.

For sensitive memories, future versions should support:

- encrypted exports
- local password protection
- desktop storage
- end-to-end encrypted sync

## Data Portability

Use the Export button to download your memories as JSON. You can import that file later or inspect it with any text editor.
