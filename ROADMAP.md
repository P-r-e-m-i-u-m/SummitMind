# Roadmap

## Version 0.1

- Local memory capture.
- Search.
- Timeline grouping.
- Pattern insights.
- Import and export.
- Open-source documentation.

## Version 0.2

- Edit existing memories. Done.
- Pin important memories. Done.
- Better tag filtering. Done.
- Monthly reflection generator. Done.
- Keyboard shortcuts.
- Print-friendly archive view.

## Version 0.3

- Local Markdown import.
- Chat transcript import.
- PDF text import.
- Voice note workflow.
- More careful pattern detection.

## Version 1.0

- Optional AI reflection mode.
- Local embedding search.
- Encrypted archive export.
- Desktop app packaging.
- End-to-end encrypted sync as an optional advanced feature.

## Long-Term Vision

SummitMind becomes a private continuity layer:

- It remembers why you made decisions.
- It shows what keeps repeating.
- It helps you reopen old ideas with context.
- It keeps your data yours.
