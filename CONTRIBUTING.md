# Contributing

Thanks for caring about SummitMind.

This project should feel calm, useful, and deeply respectful of the person using it. Please keep that spirit in mind when contributing.

## Good First Contributions

- Improve empty states.
- Add sample memory templates.
- Improve accessibility.
- Add keyboard shortcuts.
- Add tests for memory analysis helpers.
- Improve import/export validation.
- Write docs for privacy and local-first design.

## Design Direction

- Avoid guilt-heavy productivity language.
- Prefer plain words over hype.
- Make the app feel like a quiet desk, not a dashboard for surveillance.
- Keep private data private by default.

## Development

This MVP has no build step.

1. Clone the repo.
2. Open `index.html` in a browser.
3. Edit `app.js` or `styles.css`.
4. Refresh the browser.

## Pull Request Checklist

- The app still works by opening `index.html`.
- No external tracking scripts were added.
- Local data remains local unless the user explicitly exports it.
- Text fits on mobile and desktop.
- Changes are small enough to review.
